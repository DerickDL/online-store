# Online Store

## Project Description
Your task is to build a basic online store using Laravel, with the following functionality:

1. Users can create an account, log in, and log out of the system.
2. Users can browse products in the store, add them to their cart, and checkout their order.
3. Admins can add new products, update existing products, and delete products from the store.
4. Admins can view a list of orders, mark orders as shipped, and cancel orders if necessary.

## Requirements
  - Use Laravel 8 or later, and the latest stable versions of any other packages or libraries you use.
  - Use a MySQL or PostgreSQL database to store your data.
  - Implement authentication and authorization using Laravel's built-in authentication system.
  - Use Laravel's validation system to validate user input and ensure data integrity.
  - Write unit tests to test your application's functionality.
  - Use Git for version control, and provide a Git repository with your code and any necessary instructions for running your application.
